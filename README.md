#  Problema 1 — Cardápio de Lanchonete

Aplicação simples com frontend em **HTML/CSS/JS** e backend em **Node.js**.

---

##  Execução

1. Instale as dependências do backend:
   ```bash
   cd backend
   npm install

2. Inicie o servidor
   ```bash
   npm start ou npm run dev

3. Acesse no navegador 
   http://localhost:3000
